# Images du Carrousel Hero

Placez vos images de carrousel ici avec les noms suivants :

- **lab-1.jpg** - Image pour le premier slide (Réactifs de Laboratoire)
- **lab-2.jpg** - Image pour le deuxième slide (Distribution Agréée)
- **delivery.jpg** - Image pour le troisième slide (Livraison Rapide)

**Recommandations :**
- Dimensions : 1920x1080px (16:9)
- Format : JPG ou PNG
- Poids : < 500KB pour un chargement rapide
- Style : Images professionnelles de laboratoire, livraison, ou diagnostic médical

**Note :** Les images seront affichées avec un dégradé de couleur par-dessus pour assurer la lisibilité du texte. Si vous n'avez pas encore les images, le carrousel utilisera des dégradés de couleurs comme fallback.
